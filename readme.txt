# COVID19-climate

Replication material for "Ultraviolet radiation decreases COVID-19 growth rates: Global causal estimates and seasonal implications"
Authors: Tamma Carleton, Jules Cornetet, Peter Huybers, Kyle C. Meng, Jonathan Proctor  	
All data is from publicly available sources and described in Tables S1-S2 and SM section 3. Methods are described in SM section S1 and S2. Full replication code and data will be released following peer review of our data and methods.
